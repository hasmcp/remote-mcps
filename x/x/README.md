# x

X Twitter API v2

**Base URL:** `https://api.x.com/2`

**OAuth2:** ❌ Not available

## Tools (3)

- [createPost](tools/create-post.md) — Create a new post on X. At least one of text, media, or poll must be provided.
- [deletePost](tools/delete-post.md) — Delete a post on X by tweet ID
- [searchPosts](tools/search-posts.md) — Search recent posts on X. Supports operators like from:user, #hashtag, -is:retweet.
